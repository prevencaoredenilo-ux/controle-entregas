# Atualização para a V6

1. Abra o app atual e faça **Backup** dos dados.
2. Baixe e extraia o ZIP da V6.
3. No mesmo repositório do GitHub Pages, substitua todos os arquivos pelos arquivos da V6.
4. Confirme o commit.
5. Aguarde a publicação.
6. Abra o site e pressione **Ctrl + F5** no computador. No celular, feche e abra o app novamente.

## Novidades
- Novo layout completo.
- Ambiente **Operação real** e **Treinamento** separados.
- Dados de exemplo para treinamento.
- Limpeza de treinamento sem afetar dados reais.
- Apagar entregas, ciclos, KM e custos.
- Lixeira com restauração e exclusão definitiva.

## Regra de segurança
Itens apagados vão para a **Lixeira**. Só somem definitivamente quando você confirmar a exclusão permanente.
