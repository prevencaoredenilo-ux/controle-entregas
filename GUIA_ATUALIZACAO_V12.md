# Atualização para V12 — Visibilidade Total

Esta versão parte da V11 e mantém as funções já existentes. O foco é corrigir a legibilidade em toda a aplicação.

## O que foi corrigido

- Nomes de bairros em registros, tabelas, listas e seletores.
- Textos escuros sobre fundos escuros.
- Títulos, subtítulos, labels, campos, placeholders e opções de listas.
- Tabelas de entregas, ciclos, KM, custos, cadastros e análise por bairro.
- Cards operacionais, modais, pendências e telas de treinamento.
- Contraste dos botões, badges e estados.
- Ícones de data e hora em campos escuros.
- Número da compra e nome do bairro mais destacados nos registros.

## Como atualizar no GitHub

1. Faça Backup JSON no sistema atual.
2. Extraia o ZIP da V12.
3. Envie os arquivos da V12 para a raiz do mesmo repositório do GitHub Pages, substituindo os arquivos antigos.
4. Confirme o commit.
5. Aguarde a publicação.
6. Abra o site e pressione Ctrl + F5.
7. Confirme no rodapé: V12.0.0.

A V12 usa um novo nome de cache do service worker para reduzir o risco de a versão antiga continuar presa no navegador.
