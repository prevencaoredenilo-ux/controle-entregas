# NX Entregas — V5 Premium

Sistema web/PWA para gestão de entregas, ciclos, odômetro diário, custos, bairros, programadas, pendências e relatórios.

## Novidades da V5

- Novo layout premium em todo o web app.
- Cards, menu, tabelas, modais e experiência mobile redesenhados.
- Botão **Gerenciar entregas** em cada ciclo.
- Adição e remoção de entregas de um ciclo já criado.
- Ciclos abertos podem ser ajustados livremente.
- Ciclos fechados podem ser corrigidos em modo avançado com confirmação explícita.
- Indicadores são recalculados automaticamente após mudanças no ciclo.
- Histórico da entrega registra quando foi adicionada ou removida de um ciclo.
- Mantida a lógica correta: KM inicial no começo do expediente e KM final no fim do expediente; KM por ciclo e por entrega são médias derivadas do KM diário real.
