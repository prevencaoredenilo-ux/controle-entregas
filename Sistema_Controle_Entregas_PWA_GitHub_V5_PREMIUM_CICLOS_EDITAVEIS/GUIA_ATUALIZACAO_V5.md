# Atualização para V5 Premium

1. Abra o app atual e faça um **Backup JSON**.
2. Baixe e extraia o ZIP da V5.
3. No repositório atual do GitHub, use **Add file > Upload files**.
4. Envie todos os arquivos da V5 para a raiz, substituindo os antigos.
5. Confirme o commit.
6. Aguarde o GitHub Pages publicar.
7. Abra o site e use `Ctrl + Shift + R` no PC para forçar atualização.
8. No celular, feche e abra novamente o PWA. Se ainda aparecer versão antiga, abra pelo navegador e atualize.

## Teste recomendado

1. Crie duas entregas.
2. Monte um ciclo com uma delas.
3. Abra **Ciclos > Gerenciar entregas**.
4. Adicione a segunda entrega.
5. Remova a primeira.
6. Confira se o total do ciclo e os indicadores foram recalculados.
