# Controle de Entregas • Nilo Supermercado — V11

Versão focada em simplicidade operacional e prevenção de erros.

## Principais mudanças
- Central de Operação como tela inicial.
- Próxima ação recomendada automaticamente.
- Número da compra em destaque máximo.
- Linha do tempo visual: Compra → Saída → Entregue ao cliente → Retorno.
- Central de Erros e Pendências por prioridade: Crítica, Atenção e Informativa.
- Encerramento inteligente do dia com bloqueios para ciclo aberto, entrega em rota e KM pendente.
- Prevenção reforçada de duplicidade de número de compra e cupom.
- Tela de montar saída prioriza atrasadas e mais antigas.
- Dashboard enxuto com 5 indicadores essenciais e painéis separados por assunto.
- Mantém modo treinamento, lixeira, ciclos editáveis, KM diário, custos, relatórios e funcionamento offline.

Os dados antigos continuam compatíveis; faça backup JSON antes de atualizar.
