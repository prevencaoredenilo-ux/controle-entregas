# Atualização para V9 — Paleta Nilo Supermercado

1. Faça Backup JSON no app atual.
2. Extraia o ZIP da V9.
3. No mesmo repositório do GitHub, substitua os arquivos da raiz pelos da V9.
4. Faça commit.
5. Aguarde o GitHub Pages publicar.
6. Abra o site e use Ctrl+Shift+R.
7. Se ainda aparecer a versão antiga, abra em janela anônima ou limpe o cache do site/PWA.

A V9 mantém os dados compatíveis com a V8 e altera principalmente identidade visual, contraste e foco.
