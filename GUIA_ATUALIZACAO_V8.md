# Atualização para V8 — Foco Médio-Escuro

1. Abra o app atual e faça **Backup JSON**.
2. Extraia o ZIP da V8.
3. No mesmo repositório GitHub Pages, substitua os arquivos da raiz pelos da V8.
4. Confirme o commit.
5. Aguarde a publicação do GitHub Pages.
6. Abra o site e use `Ctrl + Shift + R` ou uma janela anônima se a versão antiga aparecer por cache.

A V8 mantém a estrutura de dados e as funções da V7; a mudança principal é visual e de hierarquia para deixar o uso mais rápido, concentrado e menos confuso.
