# Guia de atualização • V10

1. Faça um **Backup JSON** no app atual.
2. Extraia o ZIP da V10.
3. No mesmo repositório do GitHub Pages, substitua os arquivos da raiz pelos arquivos desta versão.
4. Faça o commit.
5. Aguarde a publicação.
6. Abra o app e faça `Ctrl + F5` ou teste em janela anônima para forçar a atualização do cache.

A V10 mantém o mesmo banco IndexedDB e foi preparada para preservar os dados existentes no mesmo navegador e endereço do app. Mesmo assim, o backup antes da atualização é obrigatório por segurança.
